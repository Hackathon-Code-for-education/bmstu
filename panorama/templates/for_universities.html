Для университетов

На данной странице вы можете зарегистрировать свой университет. Для этого выполните ряд действий.
  1. Зарегистрируйтесь на сайте, как представитель вуза (у вуза может быть только один аккаунт представителя)
  2. Авторизуйтесь в аккаунте представителя
  3. Перейдите на страницу "Университетам"
  4. Нажмите "Зарегистрировать вуз"
Для проверки вуза будет предложено несколько вариантов проверки документов. Данный этап можно завершить в настройках вуза.
